# codingStandard

编程规范

## JavaScript

### 缩进

使用两个空格，不要使用 Tab 键

### 函数名后不使用空格

```
function foo() {
  return "bar"
}
```

### 括号内两侧不使用空格

```
if (true) {

}

for (var i = 0; i < 10; i++) {

}
```

### 条件语句关键字后空一格

```
if (true) {

}

while (true) {

}

switch (sth) {

}
```

### 逗号和冒号后空一格

```
var arr = ['a', 'b', 'c']

var obj = {
  key: value
}
```

### 行末逗号

```
var obj = {
  foo: 1,
  bar: 'a',
  baz: 3.45
}
```
